# WeatherGDSC
A ReactJS App built for GDSC recruitments 2023.

## TechStack
- ReactJS
- TailwindCSS
- NextUI
- OpenWeatherMap API

# How to run locally
1. Clone this repository
2. Run `npm install`
3. Run `bash startup.bash` and paste in your OpenWeatherMap API key when prompted.
4. Run `npm start`
5. Go to `http://localhost:3000`
